# CHIP-8 Emulator (Java)

A  CHIP-8 emulator written in Java using Swing for rendering and input handling.

---

## Features

- 64x32 monochrome display
- 16-key hexadecimal keypad support
- CHIP-8 instruction set implementation
- Basic sound timer (system beep)
- ROM loader via file system

---

## Controls

Mapped to a standard CHIP-8 keypad layout:

1  2  3  4  ->  1  2  3  C

Q  W  E  R  ->  4  5  6  D

A  S  D  F  ->  7  8  9  E

Z  X  C  V  ->  A  0  B  F

---

## Running the Emulator

1. Build the project using your  Java build tool of choice (or IDE).
2. Run the `Main` class.
3. browse for ROMs through the file menu

---

## Roms

ROMs are not provided in this repository. Feel free to try some out from the [CHIP-8 Archive](https://johnearnest.github.io/chip8Archive/).
